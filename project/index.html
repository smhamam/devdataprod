Fuctionality

The application allows the user to approximate using polynomial terms one of the follwing functions:

  • Sine
  • Cosine
  • Logarithm

Functions

curve(log(x), 0, 2*pi, col="red")
curve(sin(x), add = TRUE, col = "blue")
curve(cos(x), add = TRUE, col = "green")
